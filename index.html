Why MNemo

The SQLite of agent memory

Every other memory solution is cloud-hosted, tied to a heavyweight database, or a raw vector store with no agent-native abstractions. MNemo fills that gap.

ūüďĄ

Single file

The entire memory store ‚ÄĒ vectors, metadata, indexes, WAL ‚ÄĒ lives in one .mnemo file. Copy it, back it up, email it, delete it.

ūüĒź

Encrypted at rest

AES-256-GCM page-level encryption with Argon2id key derivation. Two-tier key hierarchy means rekey never rewrites bulk data.

ūüß†

Agent-native memory

Episodic, semantic, procedural, and working memory types with multi-signal recall ‚ÄĒ similarity blended with recency, importance, and frequency.

‚ö°

Sub-linear search

IVF + Product Quantization index for sub-linear recall at scale. Exact brute-force always available as ground truth.

ūüíĺ

Crash-safe WAL

Write-ahead log with single-fsync commit. Kill the process mid-write ‚ÄĒ the database recovers cleanly on next open.

‚Ź™

Point-in-time recovery

Every flush creates a snapshot. Roll back to any past transaction ‚ÄĒ restores are themselves crash-safe and reversible.

Agentic-first

Quick Start

Ten lines to persistent memory

use mnemo::{Mnemo, MnemoConfig, Memory, MemoryType, RecallRequest};

fn main() -> mnemo::Result<()> {
    let cfg = MnemoConfig { dimensions: 768, ..Default::default() };
    let mut db = Mnemo::create("agent.mnemo", "correct horse battery", cfg)?;

    db.remember(
        Memory::new("the user prefers dark mode", MemoryType::Semantic, vec![0.1, 0.2, 0.9])
            .with_agent("assistant-1")
            .with_importance(0.8),
    )?;
    db.flush()?; // durable + crash-safe

    for hit in db.recall(&RecallRequest::new(vec![0.1, 0.2, 0.9]).top_k(5))? {
        println!("{:.3}  {}", hit.score, hit.memory.content);
    }
    Ok(())
}

Four types of memory

Structured memory types mirror how agents actually think ‚ÄĒ not just a bag of embeddings.

ūüďĖ

Episodic

What happened ‚ÄĒ conversation turns, events, interactions. Consolidated from working memory at session close.

Semantic

What is known ‚ÄĒ facts, entities, relationships, beliefs. Long-lived knowledge with contradiction detection.
